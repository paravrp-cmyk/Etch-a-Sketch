# Etch-a-Sketch

A browser-based sketching board built as part of **The Odin Project** curriculum. This project demonstrates dynamic DOM manipulation with JavaScript and responsive grid alignment using CSS Flexbox.

## Features

* **Dynamic Grid Generation:** Automatically creates a customizable pixel grid (up to 100x100) using JavaScript loops.
* **Progressive Darkening Effect:** Every time the mouse hovers over a grid square, its opacity increases by 10%, creating a realistic shading pencil effect.
* **Cozy Cozy Palette:** Customized with a warm, eye-friendly coffee and chocolate color theme.
* **Responsive Layout:** Grid squares dynamically calculate their size using `calc()` to fit perfectly within the workspace.

## Technologies Used

* HTML5
* CSS3 (Flexbox, Custom properties)
* JavaScript (DOM Manipulation, Event Listeners)
